# API - Bank Reconciliation
**Base Path:** `/api/v1/bank_reconciliation`

## REST Endpoints
- `POST /api/v1/bank_reconciliation` - Create
- `GET /api/v1/bank_reconciliation` - List
- `GET /api/v1/bank_reconciliation/{id}` - Get by ID
- `PUT /api/v1/bank_reconciliation/{id}` - Update
- `DELETE /api/v1/bank_reconciliation/{id}` - Delete
