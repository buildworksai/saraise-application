# API - Budget Management
**Base Path:** `/api/v1/budget_management`

## REST Endpoints
- `POST /api/v1/budget_management` - Create
- `GET /api/v1/budget_management` - List
- `GET /api/v1/budget_management/{id}` - Get by ID
- `PUT /api/v1/budget_management/{id}` - Update
- `DELETE /api/v1/budget_management/{id}` - Delete
