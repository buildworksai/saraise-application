# API - Regional
**Base Path:** `/api/v1/regional`

## REST Endpoints
- `POST /api/v1/regional` - Create
- `GET /api/v1/regional` - List
- `GET /api/v1/regional/{id}` - Get by ID
- `PUT /api/v1/regional/{id}` - Update
- `DELETE /api/v1/regional/{id}` - Delete
