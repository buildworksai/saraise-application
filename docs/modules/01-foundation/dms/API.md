# API - Document Management
**Base Path:** `/api/v1/dms`

## REST Endpoints
- `POST /api/v1/dms` - Create
- `GET /api/v1/dms` - List
- `GET /api/v1/dms/{id}` - Get by ID
- `PUT /api/v1/dms/{id}` - Update
- `DELETE /api/v1/dms/{id}` - Delete
